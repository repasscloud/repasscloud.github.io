Platform Engineer

Role Overview

We are seeking a Platform Engineer to help design and build internal platforms, developer tooling, and infrastructure systems used by client environments.

  • Implement infrastructure as code
  • Build internal tooling and automation
  • Improve deployment pipelines
  • Monitor and optimise system performance

Required Experience

  • Experience with cloud platforms
  • Infrastructure as code
  • Automation tooling
  • Understanding of distributed systems

Nice to Have

  • Kubernetes or container platforms
  • CI/CD systems
  • Monitoring and observability platforms

Location

Remote
