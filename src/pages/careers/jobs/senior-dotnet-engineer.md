**Type:** Contract · **Location:** Remote (Australia / International) · **Time zone:** AEST overlap required

## About the Role

We're looking for a Senior .NET Engineer to design and build production-grade backend systems, APIs, and integrations for our clients. You'll work autonomously within a small, technically capable team — owning your domain end to end.

The hiring decision for this role rests with the **interview panel**.

## What You'll Work On

- Designing and building .NET Web APIs and backend services
- Cloud infrastructure and deployment (Azure / AWS)
- Database modelling and query optimisation
- Integrations with third-party systems and APIs
- Technical documentation and architecture decisions

## What We're Looking For

- Strong .NET / C# experience with production deployments
- Solid API design fundamentals (REST, versioning, error handling)
- Experience with relational databases (SQL Server, PostgreSQL)
- Cloud experience (Azure preferred)
- Clear written communication
- Able to work independently across time zones

## Nice to Have

- Experience with containerisation (Docker, Kubernetes)
- CI/CD pipeline experience (GitHub Actions, Azure DevOps)
- Prior consulting or client-facing delivery experience

## How to Apply

Contact us at [contact page](/contact) and include:

- Your resume or CV
- GitHub profile or portfolio
- A short introduction video (2–5 minutes) telling us about yourself and your work

We review every application. You will receive a response.
