## About CveInfo

CveInfo is a command-line utility for querying Common Vulnerabilities and Exposures (CVE) information. It was built to support security operations and patch management workflows where engineers needed quick, scriptable access to CVE data without relying on a browser or a separate platform.

### What It Does

CveInfo allows users to look up CVE records from the command line, making it straightforward to integrate vulnerability lookups into scripts, automated patching pipelines, ticketing workflows, and security review processes.

### Why It Was Built

Manually looking up CVEs during patch review, incident response, or audit processes is slow and error-prone when done through a web browser. CveInfo was built to make that lookup fast and scriptable — usable inside PowerShell scripts, CI/CD pipelines, or any workflow that benefits from command-line access to vulnerability data.

### Technology

CveInfo was built in C# on .NET, consistent with RePass Cloud's primary tooling stack.
