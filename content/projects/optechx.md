## Service Status

OptechX is **retired** and is no longer an active service operated by RePass Cloud Pty Ltd. No customers are live on RePass Cloud operated OptechX infrastructure. No new subscriptions are offered and no renewals are processed. The hosted service is no longer operational and support is not provided.

This page is retained for historical reference only.

## About OptechX

OptechX was a Software-as-a-Service platform designed to simplify the process of building custom Windows Standard Operating Environment (SOE) images and preparing them for deployment through common enterprise tooling.

OptechX aimed to reduce the overhead of managing package dependencies and application versions, applying security baselines and best practices, keeping images updated with patches and releases, and producing consistent image outputs for different device models and deployment targets.

OptechX was built to integrate with common enterprise deployment approaches including OEM channels and Windows deployment frameworks. It provided a web-based interface to streamline SOE image design and automation without requiring organisations to manage complex build pipelines internally.

## Open Source Availability

OptechX has been released as open source and is available on GitHub. RePass Cloud does not operate the OptechX hosted service. Any use of the source code is at your own risk and subject to the licence terms in the repository.
