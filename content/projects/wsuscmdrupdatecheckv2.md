## About WSUS Commander Update Check V2

WSUS Commander Update Check V2 is a Windows desktop utility for checking and managing Windows Server Update Services (WSUS) update status on client machines. It was built to give IT administrators and engineers a cleaner, more reliable way to interact with WSUS update state than the tooling available at the time.

The V2 designation reflects a significant rebuild from an earlier version — designed to be more robust, better structured, and more useful in practice.

### What It Does

WSUS Commander Update Check V2 allows IT staff to query the WSUS update status of a Windows machine, review pending or available updates, and interact with the WSUS update pipeline from a purpose-built interface rather than relying on built-in Windows Update tooling or command-line workarounds.

### Who It Was Built For

It was built for IT administrators managing Windows environments with WSUS-based patch management — particularly in organisations where the built-in Windows Update tooling provided insufficient visibility or control over update state at the endpoint level.

### Technology

WSUS Commander Update Check V2 was built in C# on .NET, with a structured codebase split across Helpers, Models, and Services layers. It ships with an Inno Setup-based installer for straightforward deployment in Windows environments.
