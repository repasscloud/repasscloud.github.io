## About LockWorkstation

LockWorkstation is a lightweight Windows utility that locks the current workstation programmatically. It was built to support scenarios where locking a machine needs to be triggered from a script, scheduled task, remote management workflow, or automated process — rather than relying on a keyboard shortcut or manual user action.

### What It Does

LockWorkstation invokes the Windows workstation lock function from the command line, making it callable from PowerShell scripts, batch files, task schedulers, remote management tools, and any other automation context that can execute a binary.

### When It Was Used

LockWorkstation was useful in IT environments where workstation security policies needed to be enforced programmatically — for example, locking machines at the end of a scheduled session, as part of an automated logout sequence, or as a triggered action from a remote management or monitoring system.

### Technology

LockWorkstation was built in C# on .NET with PowerShell build tooling, consistent with RePass Cloud's standard Windows tooling approach.
