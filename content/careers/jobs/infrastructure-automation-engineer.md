## Role Overview

This role focuses on designing and implementing automation systems that reduce operational overhead and improve system reliability.

Engineers in this role build tools, workflows, and infrastructure automation that allow organisations to operate systems efficiently.

## Responsibilities

- Build infrastructure automation
- Develop workflow and scheduling systems
- Automate operational tasks
- Improve system reliability
- Optimise infrastructure costs

## Required Experience

- Strong scripting or programming experience
- Infrastructure automation experience
- Understanding of cloud platforms
- Experience with Linux environments

## Nice to Have

- Container platforms
- Serverless architectures
- Infrastructure cost optimisation

## Location

Remote
